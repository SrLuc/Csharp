# Questões:
Enunciado para todas as questões: De acordo com o tema que foi selecionado para o seu sobrenome, crie um sistema que:

01.  Possua pelo menos 4 classes (pelo menos uma herança), excluindo a classe program; OK
02.  Possua seus respectivos atributos (pelo menos 3 para cada classe, excluindo as heranças, que devem ter pelo menos 1 atributo) com seus tipos (identificados corretamente para armazenar a natureza do valor do atributo); OK
03.  Todos os atributos devem possuir o encapsulamento de dados; OK
04.  Cada classe deve possuir pelo menos 1 construtor (excluindo o construtor padrão vazio); OK
05.  Todo o projeto deve possuir pelo menos 6 métodos ao todo, distribuídos ao longo de todo o projeto (Exemplo.: Uma classe pode ter 3, outra 2, outra 1 e as demais nenhum método) – Excluindo os construtores; OK
06.  Pelo menos um dos métodos deve possuir sobrecarga (podendo ser um dos 6 métodos do projeto); OK
07.  Pelo menos um dos construtores deve possuir sobrecarga; OK
